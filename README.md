# PINN이 무엇인가?

PINN, Physical informed Neural Network 는 일반적인 데이터에만 의존하는 것이 아닌 실제 물리 법칙의 수식을 적용한 물리 정보 신경망이다.   

이 PINN은 일반적인 신경망과 다르게 실제 수식을 따르게 되므로 다른 모델들 보다 특정 환경을 설명하는데 특화되어있으며, 한 번 구축해둔 모델은 일반적인 수치해석적인 방식으로 ODE를 푸는 것보다 매우 빠르게 값을 내놓을 수 있다는 장점을 가진다.

이 페이지에서는 이런 PINN을 직접 만들어 보고 어떤 문제를 풀 수 있는가를 직접 시험해보며

이후 전자기학1과 전자기학2의 다양한 문제들을 실제로 풀어보고자 한다.
